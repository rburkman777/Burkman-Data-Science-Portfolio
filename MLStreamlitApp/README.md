# Machine Learning Streamlit App

## Project Overview
This app aims to illustrate some of the basics of machine learning for its users and give them access to some machine learning models.
The user will have the option to use either a decision tree model or a linear regression model on built-in or self-selected data. The user will 
then be able to customize hyperparameters to discover how changes in the mechanics of the model can impact model performance.

### How to Run the App Locally
1) Go to the MLStreamlitApp folder contained in this portfolio
2) Open the streamlit_machine_learning_app in VSCode
3) Go to the terminal of the file and make sure that the directory is set for the folder you opened it in (you can use the "cd" function to change this)
4) Type "streamlit run streamlit_machine_learning_app in the interface of the terminal. A link should be outputted
5) You should be taken to the app on a browser. If that doesn't work, copy and paste the second link it provides into a new browser

## App Features 
Two models were used: Linear regression and decision trees. See below for how you can tune hyperparameters on both models to explore different outcomes:

Decision Trees: You can select this from among the model options at the top of the app. Then, once you've selected it, you can tune parameters such
